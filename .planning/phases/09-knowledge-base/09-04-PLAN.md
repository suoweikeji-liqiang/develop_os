  <name>Task 1: History embedding utility</name>
  <files>
    src/server/ai/rag/history.ts
  </files>
  <action>
    Create `src/server/ai/rag/history.ts`:

    Import `{ chunkText, embedAndStore } from './embed'`
    Import `{ prisma } from '@/server/db/client'`

    Export `extractTextFromConversationContent(content: unknown): string`:
    - ConversationMessage.content is a UIMessage parts array stored as Json
    - Parse it: `const parts = Array.isArray(content) ? content : (content as any).parts ?? []`
    - Extract text: filter parts where `part.type === 'text'`, join `part.text` values with '\n'
    - Return the joined string, trimmed

    Export `embedConversationMessage(message: { id: string; requirementId: string; role: string; content: unknown }): Promise<void>`:
    - Only embed assistant messages (decisions/insights): `if (message.role !== 'assistant') return`
    - Extract text from content using `extractTextFromConversationContent(message.content)`
    - Skip if text is empty or too short (< 50 chars)
    - Chunk the text: `const chunks = chunkText(text)`
    - Call `embedAndStore(chunks, 'history', message.requirementId, { sourceName: \`conversation:\${message.id}\`, messageId: message.id })`

    Export `backfillConversationHistory(): Promise<{ processed: number; skipped: number }>`:
    - Fetch all assistant ConversationMessage rows: `prisma.conversationMessage.findMany({ where: { role: 'assistant' } })`
    - For each: check if any KnowledgeChunk exists for this message (via metadata.messageId) to avoid duplicates
      - Use: `prisma.$queryRaw\`SELECT id FROM "KnowledgeChunk" WHERE metadata->>'messageId' = ${msg.id} LIMIT 1\``
      - If chunk exists: increment skipped count, continue
      - If not: call `await embedConversationMessage(msg)`, increment processed count
    - Return `{ processed, skipped }`
  </action>
