<task type="auto">
  <name>Task 1: Install microdiff and build layer-aware diff engine</name>
  <files>src/lib/diff/structured-diff.ts</files>
  <action>
1. Install microdiff: `npm install microdiff`

2. Create src/lib/diff/structured-diff.ts with:

Types:
- DiffType = 'CREATE' | 'CHANGE' | 'REMOVE'
- LayerDiffEntry: { path: string[], type: DiffType, oldValue?: unknown, newValue?: unknown, label: string }
- StructuredDiff: { goal: LayerDiffEntry[], assumption: LayerDiffEntry[], behavior: LayerDiffEntry[], scenario: LayerDiffEntry[], verifiability: LayerDiffEntry[], summary: { added: number, changed: number, removed: number } }

Functions:
- computeStructuredDiff(oldModel: FiveLayerModel, newModel: FiveLayerModel): StructuredDiff
  - Iterates over the 5 layer keys: goal, assumption, behavior, scenario, verifiability
  - For each layer, calls microdiff's diff(oldModel[layer], newModel[layer])
  - Maps each raw diff entry to a LayerDiffEntry with a human-readable label via buildChangeLabel()
  - Tallies summary counts (added/changed/removed)

- buildChangeLabel(layer: string, change: Difference): string
  - Generates bilingual-friendly labels based on layer + path. Examples:
    - goal.summary changed → "修改了目标概述"
    - assumption.items[2] created → "添加了假设 #3"
    - scenario.normal[0].steps[1] changed → "修改了正常场景步骤"
    - behavior.actions[1] removed → "删除了行为动作 #2"
  - Use a label map keyed by layer name for the Chinese labels
  - Fall back to path-based description for unexpected paths

Export: computeStructuredDiff, StructuredDiff, LayerDiffEntry, DiffType
  </action>
  <verify>
    <automated>npx tsc --noEmit 2>&1 | head -20</automated>
  </verify>
  <done>microdiff installed. computeStructuredDiff produces layer-grouped diffs with semantic Chinese labels for all five layers. TypeScript compiles.</done>
</task>
